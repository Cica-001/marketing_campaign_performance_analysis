Simulated 2021 marketing campaign dataset with one row per campaign. Includes impressions, clicks, conversion rate, acquisition cost, ROI, campaign type, channel, and location. Designed for marketing performance analysis, Excel dashboards, and business intelligence practice.

# Marketing Campaign Performance Analysis (Excel)

## Overview
This project analyzes marketing campaign performance using Excel to evaluate key metrics such as impressions, clicks, conversion rate, acquisition cost, and ROI. The goal is to understand campaign effectiveness across channels, locations, and campaign types.

## Dataset
The dataset contains simulated marketing campaign data, including:
- Campaign type and channel used
- Target audience and customer segment
- Impressions, clicks, and conversion rate
- Acquisition cost and ROI
- Location and time period

## Tools Used
- Microsoft Excel
- Pivot Tables
- Basic Excel formulas
- Dashboard design

## Analysis Performed
- Overall campaign performance summary
- Weekly trend analysis for impressions, clicks, conversion rate, and ROI
- Performance comparison by:
  - Campaign type
  - Marketing channel
  - Location

## Key Insights
- Campaign performance remains relatively consistent across channels
- Display and Search campaigns show slightly higher average ROI
- Acquisition costs remain stable across time periods
- Similar ROI values across segments suggest the need for deeper efficiency metrics

## Limitations
- ROI and conversion rate are pre-calculated in the dataset and were analyzed using averages
- Weighted metrics and revenue-based ROI calculations were not included
- Dataset is limited in size and scope

## Next Steps
- Introduce efficiency metrics such as CPC, CTR, and CPM
- Apply weighted average calculations for ROI and conversion rate
- Expand analysis using Power BI or SQL for deeper insights
